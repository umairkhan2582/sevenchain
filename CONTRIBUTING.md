# Contributing to Seven Chain

  Thanks for your interest in contributing! Here's how to get started.

  ## Ways to Contribute

  - **Examples** — add a new `/examples/your-dapp` folder with a working contract + deploy script
  - **Docs** — fix typos, improve explanations, add tutorials
  - **Bug reports** — open an issue with steps to reproduce
  - **Feature requests** — open an issue or start a GitHub Discussion

  ## Submitting a PR

  1. Fork this repo
  2. Create a branch: `git checkout -b feat/your-feature`
  3. Make your changes
  4. Open a PR with a clear description of what you built and why

  ## Example Standards

  Each example in `/examples/` should include:
  - `contracts/` — Solidity source files
  - `scripts/deploy.ts` — Hardhat deploy script
  - `README.md` — what it does, how to deploy, expected output

  ## Questions?

  Open a [GitHub Discussion](https://github.com/umairkhan2582/sevenchain/discussions) or join [Telegram](https://t.me/SevenBlockChain).
